# Prism Demo GIF

The README demo GIF is generated by the screenshot pipeline:

```sh
./scripts/screenshot.sh
```

The command builds Prism, creates a sandboxed fake repository, runs the scripted terminal demo, and writes `docs/prism-demo.gif`.

Useful options:

- `--check` validates setup without writing `docs/prism-demo.gif`.
- `--keep` keeps the sandbox under `target/screenshots/` for debugging.
- `--skip-build` reuses `target/release/prism`.
- `--frames` keeps intermediate frame output when supported by the recording pipeline.
- `--output <path>` writes the GIF somewhere else.

All temporary demo state is kept under `target/screenshots/` and can be deleted after debugging.
